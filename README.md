Script feito para pegar as cordenadas de um retangulo na tela e das cores que tem dentro dele util para quem for criar bots.

# Como usar

Execute o script no terminel e então use a tecla shift pressionando o botão esquerdo do mouse para buscar as coordenadas do local que queira.

<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/4c30051a-c412-45d9-b068-e647f9b346c3" />

Apos isso pressione esc e abra novamente o terminal, la tera a coordenadas do local do retangulo e a cor que esta dentro dele.

<img width="707" height="498" alt="image" src="https://github.com/user-attachments/assets/4e9617ca-9e20-48ca-8144-e3de32dfeac4" />

Na imagem a cima esta marcado a cor e a coordenada como exemplo:

Cor em (226, 62): #000000
Coordenadas: (226, 62, 414, 68)
